phase. They are kept for provenance but are **not** part of any build and are
**not** the canonical manuscript. Nothing in the active codebase or the
canonical drafts references them; they cross-reference only one another.

For the actual paper, see `paper/README.md` and the canonical drafts
(`paper/OPTICS_DRAFT.md`, `paper/UNIFIED_PAPER_DRAFT.md`).
