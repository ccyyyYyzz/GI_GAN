## 0. Setup and notation

Fix a single linear forward operator `A ‚ąą ‚ĄĚ^{m√ón}` with `m < n` (undersampled), acquiring
`y = A x‚čÜ + ő∑`. Let `A‚Ä† ` be the Moore‚ÄďPenrose pseudo-inverse, and define the two orthogonal projectors

```
P_R = A‚Ä†A        (range / row-space projector,   rank m‚Ä≤ = rank A ‚Č§ m)
P_0 = I ‚ąí A‚Ä†A     (null-space projector,           rank n ‚ąí m‚Ä≤ ‚Č• n ‚ąí m > 0)
A P_0 = 0.
```

The null space `Null(A) = range(P_0)` is nontrivial because `m < n`. A *reconstructor* is any (possibly
learned, possibly nonlinear) map `f: ‚ĄĚ^m ‚Üí ‚ĄĚ^n`. We call `f` **feasible** at `y` if `A f(y) = y` (in the
noiseless idealization) or `‚ÄĖA f(y) ‚ąí y‚ÄĖ ‚Č§ ‚ÄĖő∑‚ÄĖ` (noisy). Ghost imaging / single-pixel imaging is exactly this
regime with a *single* sensing operator `A` (the fixed pattern ensemble) and `m ‚Č™ n`
[R018 ‚ÄĒ Gibson, Johnson & Padgett, *Opt. Express* 2020; R016 ‚ÄĒ Shimobaba et al. 2017].

## 1. Proposition (single-operator non-identifiability of null content)

> **Proposition 1 (Null coordinates are non-identifiable from `y` alone).**
> Let `A ‚ąą ‚ĄĚ^{m√ón}`, `m < n`, and let `y = A x‚čÜ` (noiseless). Then for **every** vector `q ‚ąą range(P_0)` the
> image `x_q := x‚čÜ + q` satisfies `A x_q = y`. Consequently:
>
> **(i) (Feasible set = an affine null coset.)** The set of measurement-consistent images is exactly
> `ūĚíě(y) = A‚Ä†y + range(P_0)`, an affine subspace of dimension `n ‚ąí rank A ‚Č• n ‚ąí m > 0`.
>
> **(ii) (No estimator can identify the null coordinates.)** The map `x ‚Ü¶ P_0 x` is **not identifiable** from
> `y`: for any two ground truths `x‚čÜ, x‚čÜ‚Ä≤` with `A x‚čÜ = A x‚čÜ‚Ä≤` but `P_0 x‚čÜ ‚Č† P_0 x‚čÜ‚Ä≤`, no measurable function
> `g(y)` can equal `P_0 x` for both ‚ÄĒ the likelihood is literally constant along `range(P_0)`, so the data
> supply *zero* information about `P_0 x`. This holds **regardless of the reconstructor** `f`, learned or not,
> and independently of image quality.
>
> **(iii) (Noisy version.)** With `y = A x‚čÜ + ő∑`, the log-likelihood `‚Ąď(x; y)` depends on `x` only through
> `A x = A P_R x`. Hence `‚ąā‚Ąď/‚ąā(P_0 x) ‚Č° 0` and the Fisher information for the null coordinates is exactly the
> zero matrix; the null block of the parameter is a *flat direction* of the likelihood for every noise model
> whose density depends on `x` only via `Ax`.

The certified quantity is therefore the **range coset label** `P_R x = A‚Ä†y` (identified, and audited per-mode
by our certificate at contraction `őĽ/(őĽ+ŌÉ_i¬≤)` ‚ÄĒ see P2). The null coordinate `P_0 x` is a **free
normalization** that any reconstructor selects by an *unstated prior*, not by the measurement.

### Proof-sketch

**(i)** `A x_q = A x‚čÜ + A q = y + A P_0 v = y` since `A P_0 = 0` for any `q = P_0 v`. Conversely if `A x = y`
then `A(x ‚ąí A‚Ä†y) = y ‚ąí AA‚Ä†y = 0`, so `x ‚ąí A‚Ä†y ‚ąą Null(A) = range(P_0)`; thus `x ‚ąą A‚Ä†y + range(P_0)`. The
decomposition `x = A‚Ä†y + P_0 x` is orthogonal, so the coset has dimension `dim Null(A) = n ‚ąí rank A`. ‚ąé

This is precisely Proposition 1 of Equivariant Imaging ‚ÄĒ *any* `f(y) = A‚Ä†y + v(y)` with `v(y) ‚ąą Null(A)` is
measurement-consistent, so "consistency alone gives no information on the null-space component"
[R023 ‚ÄĒ Chen, Tachella & Davies, ICCV 2021]. HalluGen names the same object its *extrinsic* class
(measurement-consistent yet semantically wrong) [R002 ‚ÄĒ Kim et al. 2025].

**(ii)** Because `A x_q = y` for all `q ‚ąą range(P_0)`, the two ground truths `x‚čÜ, x‚čÜ‚Ä≤` induce identical
observation distributions (identical `y`), i.e. they are **observationally equivalent**. Identifiability of a
functional `T(x)` requires `T` to be constant on each observational-equivalence class; here the class *is* the
null coset, on which `P_0 x` is by construction non-constant. Hence `P_0 x` is unidentifiable in the exact
sense of the classical definition (a parameter is identifiable iff distinct parameter values yield distinct
observation laws) ‚ÄĒ the null coset is the *observational-equivalence set* [IP-6 cross-field cards: Koopmans‚Äď
Reiers√łl / Manski observational equivalence, Rothenberg singular-information direction, Landau below-density
lost modes]. This is exactly the "there is no information about the model in the null space of `A`" statement
[R025 ‚ÄĒ Tachella, Chen & Davies, NeurIPS 2022]; REI restates it under noise ‚ÄĒ "`f_őł(y)` can output any `x` in
`A^{-1}(y)` and still perfectly fit the training set" [R024 ‚ÄĒ Chen, Tachella & Davies, CVPR 2022]. No property
of `f` (expressivity, training data, inductive bias) can manufacture information the channel did not carry.

**(iii)** Write the density `p(y | x) = p_ő∑(y ‚ąí Ax)`. For a fixed range component, perturbing `x` by
`q ‚ąą range(P_0)` leaves `Ax` fixed, so `p(y | x + q) = p(y | x)`: the likelihood is flat along `range(P_0)`.
The Fisher information `I(x) = ūĚĒľ[(‚ąá_x ‚Ąď)(‚ąá_x ‚Ąď)ŠĶÄ]` factors through `‚ąá_x ‚Ąď = AŠĶÄ ‚ąá_{Ax} ‚Ąď`, so
`P_0 I(x) P_0 = 0` ‚ÄĒ the null block is singular with a full `(n ‚ąí rank A)`-dimensional kernel. A zero Fisher
block is the textbook signature of a locally non-identified direction (Rothenberg's singular-information
direction) [IP-6 cards]. ‚ąé

**Honesty note.** (ii)‚Äď(iii) are impossibility statements about *identifiability from `y`*, not about whether a
particular `q` is aesthetically or clinically plausible. A prior can make one `q` *preferred*; it cannot make
it *certified by the measurement*. Our dial injects a chosen `q ‚ąą range(P_0)` and labels it invented by
construction ‚ÄĒ consistent with, not in violation of, Proposition 1.

## 2. What would restore identifiability ‚ÄĒ and why GI never reaches it

Non-identifiability is a property of the *information channel*, so it is repaired only by adding information.
There are exactly three routes, and we state each precisely with its cost.

> **Theorem 2 (Restoration conditions; borrowed).** The null coordinates become identifiable iff one of:
>
> **(R1) Complementary operators (active / multi-operator sensing).** Acquire `y_g = A_g x` under `G`
> operators with *distinct* null spaces. The signal `x` is uniquely determined iff the stacked operator has
> full column rank,
> `rank[A_1; ‚Ä¶; A_G] = n`  (‚áí necessary `m|G| ‚Č• n`, i.e. `m ‚Č• n/G`)
> [R023 Thm 1: `m|G| ‚Č• n`; R025 Prop 4.1]. For identifying a `k`-dimensional signal *model* from measurements
> alone, the sharp **sufficient** condition is
> `m > k + n/G`   (Sauer‚ÄďYorke embedology; a.e. choice of `G` operators)
> [R025 Thm 4.2 ‚ÄĒ Tachella et al., NeurIPS 2022], and MNIST-CS reconstruction PSNR jumps *exactly* at this
> `m > k + n/G` red line.
>
> **(R2) Invariance / equivariance prior.** Assume the signal set is invariant under a unitary group `ūĚíĘ`; then
> each `T_g` induces a *virtual* operator `A_g = A T_g` whose range is the transformed range `T_gŠĶÄ R_A`.
> Identifiability of the null content requires the virtual family to span: the stacked
> `M = [A T_1; ‚Ä¶; A T_{|ūĚíĘ|}]` must have rank `n` (`m|ūĚíĘ| ‚Č• n`), **and** ‚ÄĒ Corollary 1 of EI ‚ÄĒ `R_A` must **not**
> be `ūĚíĘ`-invariant, else the virtual operators repeat the same range and add nothing
> [R023 Thm 1 + Cor 1; R082 ‚ÄĒ Chen et al., *IEEE SPM* 2022].
>
> **(R3) Explicit prior (Bayesian normalization).** Assume a proper prior `ŌÄ(P_0 x | y)`. Then `P_0 x` is
> *estimable* (a posterior mean/MAP exists) but its posterior is **fixed by `ŌÄ`, not updated by the data**
> along `range(P_0)` ‚ÄĒ the likelihood contributes nothing there (Prop 1(iii)). This "identifiability" is a
> definition supplied by the analyst, not a measurement fact.

**Ghost imaging sits strictly below every restoration threshold.** GI/SPI use a **single** fixed sensing
operator, so `G = 1` and `m ‚Č™ n` [R018; R016]. Then:

- **(R1) fails:** the necessary condition `m ‚Č• n/G = n` demands `m ‚Č• n`, i.e. Nyquist-complete sampling ‚ÄĒ the
  opposite of the compressive `m ‚Č™ n` regime that is the entire point of ghost imaging. The sufficient
  `m > k + n` is even further out of reach.
- **(R2) fails or smuggles:** a single GI operator carries no genuine group symmetry that yields *distinct*
  virtual ranges; assuming one is an *external* prior the single measurement cannot verify (¬ß3).
- **(R3) is a labelling choice:** it renders the null coordinate *defined*, not *measured*.

This is the **formal trigger**: because none of (R1)‚Äď(R3) supplies measurement information in the single-GI
regime, the honest move is to **govern** the null coordinate (our pre-registered dial `B`, with `A xŐā_B = y`
exact ‚ąÄ`B`) rather than to *claim recovery* of it. "Certify what you measure (`P_R x`); govern what you cannot
(`P_0 x`)."

## 3. Corollary ‚ÄĒ equivariance does not escape the barrier (it relocates the assumption)

> **Corollary 3 (Equivariance smuggles unverifiable content).** Suppose a reconstructor is regularized to be
> equivariant under an *assumed* group `ūĚíĘ`, but the assumption is spurious for the true scene (the signal is
> not actually `ūĚíĘ`-invariant, or `R_A` **is** `ūĚíĘ`-invariant so no new range is probed ‚ÄĒ EI Corollary 1). Then
> one can construct a **feasible-wrong pair** `(x‚čÜ, x‚čÜ + q)`, `q ‚ąą range(P_0)`, that is *simultaneously*
> measurement-consistent (`A(x‚čÜ+q)=y`, residual ~2e-15) **and** compatible with the assumed symmetry, so the
> equivariance regularizer cannot distinguish them. Equivariance therefore does not certify the null content;
> it only *re-selects* a `q` under an external, measurement-unverifiable prior.

*Proof-sketch.* Choose `q` in the intersection `range(P_0) ‚ą© (assumed-invariant subspace)`; this intersection
is nonempty whenever the invariant subspace is not contained in `range(P_R)`, which holds precisely when the
virtual family fails to span (EI Cor 1 non-spanning case). Both twins then satisfy the measurement identically
and the symmetry penalty identically. ‚ąé [R082 innovation-spark: "even equivariance-regularized reconstructions
have unverifiable null content"; R023 Cor 1.]

This turns the *strongest* self-supervised competitor (Equivariant Imaging) into a live P1 exhibit rather than
a threat: EI legitimately recovers null content **only** when its spanning rank condition holds ‚ÄĒ a condition
GI's single operator cannot meet.

## 5. Active-sensing connection and an experiment-design proposal

Proposition 1 says null coordinates are dark to a *fixed* `A`; Theorem 2(R1) says a *second, complementary*
operator can illuminate them ‚ÄĒ this is precisely the logic of **active / multi-operator sensing** and of the
MOI cross-operator-consistency loss [R025]. Two concrete, falsifiable designs make the trigger empirical:

**Design A ‚ÄĒ the identifiability red line, in the GI modality (single figure).**
On a GI/SPI testbed, sweep the number of *distinct* pattern operators `G` (each an independent random Hadamard/
Fourier ensemble with a different null space) at fixed per-operator budget `m`. Plot recovery error of a chosen
null coordinate `P_0 x` vs the effective sampling `m¬∑G` relative to the predicted transition `m > k + n/G`
[R025 Thm 4.2]. *Predicted result:* the recoverable-null error stays at the pseudo-inverse floor for `G = 1`
(our regime) and collapses only once `m¬∑G` crosses the red line ‚ÄĒ a direct, modality-native demonstration that
single-operator GI is below threshold, and that identifiability is *bought* by complementary operators, not by
a better reconstructor. Ground truth is used **only** to score the plot, not inside any certificate.

**Design B ‚ÄĒ active complementary probe vs. certificate.** Given a first GI acquisition `y_1 = A_1 x`, choose a
second operator `A_2` to *maximally reduce* the residual null dimension `dim(Null(A_1) ‚ą© Null(A_2))` (a greedy
row-selection / D-optimal design over the joint null space). Report, as `A_2` is added row-by-row: (i) the
shrinking joint-null dimension (identifiability gained), against (ii) our per-mode certificate contraction
`őĽ/(őĽ+ŌÉ_i¬≤)` on the newly-illuminated modes. *Point:* each row that a complementary operator adds converts a
governed (invented) null coordinate into a *certified* range coordinate ‚ÄĒ operationalizing "identifiability is
recovered by extra measurements," and drawing the exact boundary our dial governs from the region a certificate
can vouch for. This connects the impossibility trigger (P1) to the certificate (P2) to active sensing in one
protocol.

**Honesty guard on both designs.** No claim is made that a fixed single-`A` GI reconstruction certifies its null
content; the experiments *quantify the gap* and show what closing it costs (more operators / active probes),
never that quality metrics or symmetry priors close it for free.
