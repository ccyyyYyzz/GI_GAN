# HQ-02 ‚ÄĒ B-as-Guidance-Scale: One Governed Knob Sweeping a *True* Perception‚ÄďDistortion Frontier

**Status: proposed** (method + proposition + experiment design fully specified; no numbers invented; expected stand-alone gain is honestly assessed as *marginal* ‚ÄĒ see ¬ß7).

**Theme.** High-quality low-rate ghost-imaging reconstruction. This point reparameterizes the single governed scalar `B` from a *linear blend weight* between two frozen residuals into the *guidance temperature* of a null-confined sampler, so that the same one-scalar governance story now traces a genuine perception‚Äďdistortion (P‚ÄďD) curve instead of a chord between two fixed images. It keeps the exact-consistency invariant `A xŐā = y` (bought entirely inside `P‚āÄ`, `A P‚āÄ = 0`) at every value of `B`.

**Relation to HQ-01.** HQ-02 is a *governance/interface* layer on top of the HQ-01 null-confined sampler (`nds_diff`). Its quality ceiling is bounded by HQ-01: on its own, over the *static* two-point residual pair `(d_A, d_G)` that the current fusion uses, `B`-as-guidance degenerates back into the current linear dial (Proposition 2). Its independent value is (i) the cleaner single-pre-registered-scalar P‚ÄďD story and (ii) a band-dependent `B(f)` generalization that the static fusion cannot express. It is written to be droppable directly onto HQ-01's substrate.

## 1. Problem with the current dial

The locked fusion (`vqgan_detail_fusion.py::fuse`, `scalar` branch) forms

```
őī   = d_G ‚ąí d_A
xŐā_B = x0 + P‚āÄ( d_A + B¬∑őī ),      B ‚ąą [0,1],     (current)
```

with `d_A = P‚āÄ(x_A ‚ąí x0)`, `d_G = P‚āÄ(x_G ‚ąí x0)` precomputed once by `prep_residuals`, and `x_A, x_G` the frozen VQAE/VQGAN refiner outputs. Sweeping `B` moves `xŐā_B` along the **straight segment** between the two fixed endpoints `xŐā_0 = x0 + P‚āÄ d_A` (structure) and `xŐā_1 = x0 + P‚āÄ d_G` (detail). The reachable set of the dial is exactly this chord:

```
R_lin = { x0 + P‚āÄ(d_A + B(d_G ‚ąí d_A)) : B ‚ąą [0,1] } ‚äā x0 + P‚āÄ¬∑conv{d_A, d_G}.
```

This is a **1-parameter affine family with two anchors**. It cannot reach P‚ÄďD-plane points that require *new* null content beyond the affine hull of `{d_A, d_G}`; the LPIPS-vs-PSNR ordering it produces is only as rich as the two endpoints allow. The `B`-swept curve reported at the locked operating point is therefore a *segment*, not the operator's attainable P‚ÄďD frontier.

The perception‚Äďdistortion theorem (R061, Blau & Michaeli) says the *frontier itself* is a monotone convex curve that no estimator can cross, steepest exactly at the low-distortion / high-perception corner where we operate at 2‚Äď5% sampling; RDP (R062) sharpens this at low rate, where the perceptual-vs-fidelity price is most severe ‚ÄĒ precisely our regime. A dial that only interpolates two frozen points touches this frontier at two isolated locations and connects them with a chord that lies *above* (worse than) the true convex frontier everywhere strictly between the endpoints. HQ-02 replaces the chord with a trajectory that hugs the frontier.

## 2. Method: `B` as null-confined guidance temperature

### 2.1 Setup (unchanged assets)

* Exact projector `ExactRangeNullProjector` (`src/projections.py`): `data_anchor_flat(y)` returns the range anchor `A‚Āļy` (LMMSE-audited `x0` supplies its own audited range component), `null_project_flat(v)` returns `P‚āÄ v = (I ‚ąí A‚ĀļA)v`, `audit_flat(v, y)` re-pins the range to kill float drift; `A P‚āÄ = 0` exactly, so `relmeaserr` stays ‚Č§ 2e-15.
* Frozen VQGAN decoder as the `x0`-predictor of a null-confined reverse trajectory (HQ-01 substrate): `prior.model.decode_embeddings(zq)` inside `refine`. Nothing is trained.
* LMMSE anchor `x0 = hq.EmpiricalLMMSE.anchor(y)`; structured DCT+Hadamard+random operators at 2/5/10%; 64√ó64 STL10; locked val‚Üídev multi-seed gate (`cmd_canary`/`cmd_gate`).

### 2.2 Guided null increment (scalar form)

Let `{x0|t}` be the per-step clean-image estimates of the HQ-01 sampler (Tweedie posterior means from the frozen decoder; R071 supplies the `xŐā‚āÄ = E[x‚āÄ|x_t]` estimator; DDNM R026 supplies the per-step range/null rectification). Define the **anchor-referenced null increment** at each step

```
u_t = P‚āÄ( x0|t ‚ąí x0 )              # the detail the sampler wants to write into Null(A) this step
```

and scale it by the governed temperature `B`:

```
xŐā0|t = x0 + P‚āÄ( B ¬∑ u_t )         # = x0 + B¬∑P‚āÄ( x0|t ‚ąí x0 )        (HQ-02 scalar)
     = data_anchor-consistent range  +  B-tempered null detail
```

then continue the reverse update from `xŐā0|t`. A final `audit_flat(¬∑, y)` removes accumulated float drift. Because `A P‚āÄ = 0`, **`A xŐā0|t = y` for every step and every `B`** ‚ÄĒ consistency is `B`-independent by construction (contrast DPS R071 / DiffPIR R075 / ESRGAN-DNI R064/R067, whose guidance / interpolation scalars all move the *measured* subspace and lose exact consistency; see ¬ß5).

Interpretation of the knob:

* `B = 0` ‚Üí `xŐā0|t = x0` at every step ‚Üí the sampler writes **zero** null detail ‚Üí output is the audited anchor `x0` (max fidelity / min perception endpoint).
* `B = 1` ‚Üí full HQ-01 null-confined DDNM trajectory (max perception endpoint).
* `B ‚ąą (0,1)` ‚Üí a **contracted-guidance trajectory**: each step still runs the generative prior, but only a `B`-fraction of the proposed null detail is committed and the *next* step re-estimates from the tempered state. Because the trajectory is re-evaluated after each contraction, intermediate `B` traces a **curved path on the manifold**, not a straight segment between `x0` and the `B=1` sample. This is the qualitative difference from ¬ß1's chord.

This mirrors DiffPIR's per-timestep `ŌĀ_t` prior/data balance (R075) and DPS's soft-gradient scale `ő∂` (R071) ‚ÄĒ but as **one global, pre-registered scalar constrained to the null space**, so quality is dialed without ever touching the certified measured modes. It is "DNI (R067) / ESRGAN net-interp (R064) done in the right coordinates": a single slider, but applied to the *increment inside `Null(A)`* rather than to network weights, so it provably cannot perturb `A xŐā`.

### 2.3 Band-dependent guidance `B(f)` (the part the static fusion cannot do)

The current code already supports a radial band schedule (`weight_map`, `apply_band`, `radial_masks`) for the *static* residual `őī`. HQ-02 lifts this to the *sampler increment*: replace the scalar `B` by a radial temperature profile `B(f)` applied to `u_t` in the FFT domain,

```
Ňę_t = ūĚĒČ‚ĀĽ¬Ļ{ B(f) ‚äô ūĚĒČ u_t } (real part),      xŐā0|t = x0 + P‚āÄ( Ňę_t ),
```

with `B(f) = ő£_b w_b ¬∑ mask_b(f)` reusing `weight_map`/`radial_masks` verbatim. Low-frequency bands take `B(f)‚Üí` small (protect anchor structure / fidelity); high-frequency bands take `B(f)‚Üí1` (spend the perceptual budget on texture the eye scores via LPIPS, R063). This is a strict generalization: `B(f) ‚Č° B` recovers ¬ß2.2, and a two-band step recovers the current `lowpass_cutN` specs ‚ÄĒ so HQ-02 **can only match or beat** the tuned band baseline.

### 2.4 Why consistency is free (Proposition 1)

> **Proposition 1 (exact consistency of guided sampling).** For any measurable temperature field `B(f)` (including the scalar `B(f)‚Č°B`), the guided update `xŐā0|t = x0 + P‚āÄ(Ňę_t)` satisfies `A xŐā0|t = A x0 = y_audited` exactly, i.e. `‚ÄĖA xŐā0|t ‚ąí y‚ÄĖ / ‚ÄĖy‚ÄĖ ‚Č§ őĶ_float` (numerically ~2e-15), independent of `B`, of the step index, and of the frozen prior's output.

*Proof.* `x0` is range-audited so `A x0 = y` to float precision. `P‚āÄ = I ‚ąí A‚ĀļA` gives `A P‚āÄ v = (A ‚ąí A A‚ĀļA)v = 0` for all `v` (since `A A‚ĀļA = A`). Hence `A xŐā0|t = A x0 + A P‚āÄ(Ňę_t) = A x0 + 0 = y`. `Ňę_t` is a linear (FFT band-weight) map of `u_t`, so it introduces no range component that survives `P‚āÄ`. The final `audit_flat` re-projects to remove accumulated float error. ‚ąé

The governance claim ‚ÄĒ *one pre-registered scalar, selected on a firewalled gate, whose entire effect is confined to `Null(A)`* ‚ÄĒ is unchanged from the current paper; only the geometry of what `B` sweeps changes.

## 3. Architecture / where it lives in the code

No new trained parameters. Concretely:

* **New sampler arm** (from HQ-01): `nds_diff(x0, y, projector, prior, steps=25, B=‚Ä¶)` producing `x0|t ‚Üí xŐā0|t` with the ¬ß2.2 guided increment. Reuses `refine`'s `encode`/`logits_from_latent`/`quantize_from_logits`/`decode_embeddings` chain as the `xŐā‚āÄ|t` predictor.
* **Scalar-branch replacement in `fuse`.** Add a spec kind `("guided", B)` that, instead of the static `float(w)*őī`, invokes the guided trajectory and returns its final `audit_flat`ed output. The linear `("scalar", B)` branch is retained as a baseline/ablation, not deleted.
* **Band form.** Add `("guided_band", w)` reusing `weight_map`/`radial_masks` to build `B(f)`; `apply_band` already does the FFT band-weighting and is reused on `u_t`.
* **Grid + selection.** Generalize `gen_specs(k)`: the current `scalar_{b}` grid (`np.linspace(0,1,21)`) becomes the `guided_{b}` grid; the `lowpass_cutN` band grid becomes `guided_band_cutN`. `select_operating_point` and the val‚Üídev firewall are untouched ‚ÄĒ HQ-02 slots into the identical selection machinery.
* **Audit.** Every arm still ends in `projector.audit_flat` + `relative_measurement_error`, so the `relmeaserr ‚Č§ 1e-5` gate is enforced exactly as today.

## 4. Training / inference recipe

**Training:** none. Both refiners and the decoder are frozen (`refiner_ckpt`), the LMMSE anchor is fit as before, the operator is the locked structured operator.

**Inference (per image, per seed, per rate):**
1. `x0 = lmmse.anchor(y)`; `unc = lmmse.uncertainty_map()`.
2. For each candidate `B` in the pre-registered grid `{0, 0.1, ‚Ä¶, 1.0}` (val only): run `nds_diff` with `steps=25`, guided increment `xŐā0|t = x0 + P‚āÄ(B¬∑u_t)` (or band form `Ňę_t`), final `audit_flat`.
3. Score `fast_means` (LPIPS/PSNR/RMSE/RAPSD/relmeaserr) on the **val** split.
4. **Firewall:** pick a single `B*` on val by `select_operating_point` (balanced or quality_lite mode), then evaluate that one `B*` on **dev/gate** and on seeds 0/1/2. Report the *entire* `B`-swept val P‚ÄďD curve for the figure, but only the gate-selected `B*` for the headline number.

**Cost.** ~25 decoder calls per image per candidate `B` (vs 1 for the static blend). The static-fusion baseline remains available for the ablation at negligible cost since `(d_A, d_G)` are cached.

## 6. Experiment / ablation design

**E1 ‚ÄĒ Frontier geometry (the core claim).** On seed0 **val**, sweep `B ‚ąą {0,0.1,‚Ä¶,1.0}` for three arms: (a) current static linear blend (`scalar`), (b) HQ-02 guided scalar (`guided`), (c) HQ-02 guided band (`guided_band`). Plot LPIPS vs PSNR. **Predicted signature:** (a) is a chord between the two endpoints; (b)/(c) bow *toward the low-LPIPS-at-fixed-PSNR corner* (convex frontier, R061), i.e. dominate the chord strictly between endpoints. Confirm the LPIPS-vs-PSNR ordering is monotone in `B` (sanity for a governed knob).

**E2 ‚ÄĒ Gate win / no-loss.** Select `B*` on val via `select_operating_point`, evaluate on **dev/gate** at 5% for seeds 0/1/2. **Success = a `B*` that beats the locked `‚ąí32.6%` LPIPS at equal-or-smaller PSNR cost on ‚Č•2/3 seeds.** (Honest expectation: this is the *hard* bar and may not clear it stand-alone ‚ÄĒ see ¬ß7.)

**E3 ‚ÄĒ Rate stress.** Repeat E1/E2 at 2% and 10%. Prediction from RDP (R062): the frontier is steepest / the guided-vs-chord gap is *largest* at 2% (lowest rate), smallest at 10%. This is where HQ-02 should show whatever advantage it has.

**E4 ‚ÄĒ Consistency ablation (Related-Work figure).** Log `relative_measurement_error` across the full `B` sweep for HQ-02 (expect flat ~2e-15) vs an unprojected guidance variant (increment applied *without* `P‚āÄ`) to visualize drift ‚ÄĒ demonstrates Proposition 1 empirically and the ¬ß5 table's last column.

**E5 ‚ÄĒ Band vs scalar.** Compare `guided_band` (E1c) vs `guided` (E1b) selected `B*`. Since scalar is the special case `B(f)‚Č°B`, band can only match-or-beat; measures how much per-band budgeting buys.

**E6 ‚ÄĒ Substrate dependence (honesty check).** Run HQ-02's `guided` arm but feed it the *static* two-point degenerate sampler (no reverse trajectory, just `x0 ‚Üí x_G` in one shot). Prediction (Proposition 2): it collapses back onto the linear chord of E1a, confirming that HQ-02's independent value *requires* the HQ-01 generative substrate.

All arms route through the identical `fuse ‚Üí audit_flat ‚Üí score_method` path and the locked gate; `relmeaserr ‚Č§ 1e-5` is a hard precondition for every reported number.

## 7. Honest assessment of expected gain (marginal, not large)

**The gain over the *current* linear fusion is expected to be marginal on its own, and its ceiling is set by HQ-01.** Reasoning, stated plainly:

> **Proposition 2 (degeneracy without a generative substrate).** If the per-step estimate is `B`-affine in the committed detail ‚ÄĒ i.e. `x0|t` depends on the running state only through a fixed direction (as in a single static endpoint `x_G`) ‚ÄĒ then the guided trajectory `xŐā0|t = x0 + B¬∑P‚āÄ(x0|t ‚ąí x0)` collapses to the linear family `R_lin` of ¬ß1, and the HQ-02 curve coincides with the current `B`-blend chord.

*Sketch.* With a one-shot endpoint `x0|t ‚Č° x_G` independent of `B`, `xŐā = x0 + B¬∑P‚āÄ(x_G ‚ąí x0) = x0 + B¬∑d_G`, which is the current scalar branch (up to the `d_A` offset). Curvature of the HQ-02 path comes *only* from the sampler re-estimating `x0|t` from the tempered state at each step ‚ÄĒ i.e. from HQ-01's generative loop. No loop ‚áí no curvature ‚áí no gain. ‚ąé

Consequences for honesty:

* **On the static `(d_A, d_G)` pair, HQ-02 is a reparameterization, not new detail.** It does not by itself write richer null content; it only changes *how* the existing detail is committed. Any LPIPS improvement over the tuned linear `B` in that setting would be small and could be within selection noise.
* **The real lift is HQ-01's.** HQ-02's job is to give HQ-01's sampler a *governed, pre-registered, consistency-exact* interface and to expose the full frontier for the P‚ÄďD figure. The headline `‚ąí32.6% ‚Üí better` win, if it materializes, is attributable to the generative substrate; HQ-02 is the knob that lets the paper *report and control* it under the single-scalar governance story.
* **Where HQ-02 does add genuine, non-degenerate value:** (i) the band form `B(f)` (¬ß2.3) expresses content the scalar linear fusion cannot (low-freq fidelity / high-freq perception budgeting on the *sampler increment*, not a static residual) ‚ÄĒ E5 isolates this; (ii) the clean ¬ß5 differentiation (same slider as ESRGAN/DNI/DPS/DiffPIR/CodeFormer but exactly consistent for *all* `B`) is a Related-Work / Dial-section asset independent of the magnitude of the LPIPS delta; (iii) tracing the true convex frontier (E1) rather than a chord is a *correctness-of-reporting* improvement even when the endpoint numbers are unchanged.

**Bottom line.** Recommend building HQ-02 *only on top of HQ-01* (do not pursue it as a stand-alone quality play). Its expected marginal LPIPS gain over the current fusion is **small**; its expected value is in **governance, P‚ÄďD-frontier reporting, and the band-`B(f)` generalization**, not a large quality jump. This matches the deck's own risk note that HQ-02 "only lifts quality if HQ-1's sampler already wins; on its own it is a reparameterization of the dial."

## 8. Success / kill criteria

* **Ship (as governance layer):** E1 shows the guided curve strictly dominates the linear chord between endpoints on val at 2% and 5%; E4 confirms `relmeaserr` flat at ~2e-15 across the whole sweep; E5 shows band `B(f)` ‚Č• scalar. Report the frontier figure + the ¬ß5 consistency-vs-`B` ablation regardless of the headline delta.
* **Headline win (optional, HQ-01-dependent):** E2 clears the locked `‚ąí32.6%` LPIPS at equal-or-smaller PSNR cost on ‚Č•2/3 seeds via the gate-selected `B*`.
* **Kill (as stand-alone):** if E6 shows the guided arm collapses to the chord *and* HQ-01's substrate is not available, HQ-02 provides no quality lift ‚ÄĒ keep only the band-`B(f)` reporting-geometry contribution, or shelve until HQ-01 lands.
